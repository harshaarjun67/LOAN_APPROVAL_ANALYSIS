Loan Approval Analysis 💳📊

This project analyzes loan application data to identify the key factors that influence loan approval decisions. Using data analysis and visualization techniques, it explores applicant profiles, income patterns, credit history, and loan status to uncover trends and support better decision-making.

📌 Project Objectives

Clean and preprocess loan application data.
Analyze applicant demographics and financial details.
Identify factors affecting loan approval.
Explore relationships between income, credit history, and loan status.
Visualize patterns for better understanding.
🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

📂 Dataset Features

The dataset includes:

Applicant Income
Co-applicant Income
Loan Amount
Loan Term
Credit History
Gender
Marital Status
Education
Self-Employed Status
Property Area
Loan Status
📈 Key Insights
Credit history has a strong impact on loan approval.
Higher income applicants tend to have better approval chances.
Loan amount and repayment term influence approval decisions.
Property area and applicant background show noticeable trends.
Missing values and outliers significantly affect analysis.
